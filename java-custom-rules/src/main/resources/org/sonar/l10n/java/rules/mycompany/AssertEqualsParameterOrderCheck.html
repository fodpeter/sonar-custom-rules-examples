This rule detects the incorrect usage of Assert.assertEquals. The first parameter of this method is the expected value and the second parameter is the calculated value.

Noncompliant Code Example

    @Test
    public void test() {
        Assert.assertEquals(ObjectJoiner.joinString("str", "ing"), "string"); // Noncompliant
        Assert.assertEquals(Math.abs(-1), 1); // Noncompliant
    }

Compliant Solution

    @Test
    public void test() {
        Assert.assertEquals("", "");
        Assert.assertEquals("string", ObjectJoiner.joinString("str", "ing"));
        Assert.assertEquals(1, Math.abs(-1));
    }
